# Code of Conduct

We are committed to a welcoming, respectful, harassment-free community.
Contributors must communicate constructively, respect differing viewpoints,
accept responsibility for mistakes, and focus on what is best for the project.

Unacceptable behavior includes harassment, discrimination, threats, publishing
private information, or sustained disruption. Project maintainers may remove
content or participation that violates these expectations.

Report conduct concerns privately to the project maintainers through the
contact method listed in the repository security settings.
